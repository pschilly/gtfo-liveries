# Global Task Force Overlord Official Livery Pack

The follow repository was created for the purpose of providing a method for members of Global Task Force Overlord (GTFO) to both collaborate on the creation of, as well as, distribution of custom liveries for usage within our Digital Combat Simulator (DCS) community.

## Installation

Download the contents of the repo as a zip file from the [Releases Page](https://github.com/pschilly/gtfo-liveries/releases) of this repo.

To install the liveries, copy the contents of the `Liveries/` folder from the unzipped download into your DCS Saved Games folder at `C:\Users\<your username>\Saved Games\DCS\Liveries`. Replace any existing files to ensure the latest updates are installed. Check that the liveries are installed after restarting DCS.
